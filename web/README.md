## Hello There

A playful single-page greeting experience built with the Next.js App Router and Tailwind CSS. Personalize the welcome message by sharing your name, see a time-aware greeting, and pick from upbeat prompts to spark your next move.

### Local Development

```bash
npm install
npm run dev
```

### Available Scripts

- `npm run dev` – start the local development server on [http://localhost:3000](http://localhost:3000)
- `npm run build` – create an optimized production build
- `npm run start` – serve the production build
- `npm run lint` – run ESLint

### Tech Stack

- Next.js 14 (App Router)
- React + TypeScript
- Tailwind CSS
- date-fns
